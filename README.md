# 싸칙 포커

숫자 카드와 사칙연산 카드로 식을 완성해 계산 결과로 승부하는 2~4인 턴제 카드 게임입니다.

## 실행

로컬 테스트 서버를 켠 뒤 브라우저에서 접속합니다.

```bash
python -m http.server 4173
```

```text
http://127.0.0.1:4173/index.html
```

Firebase 설정이 없어도 `한 기기에서` 모드로 로컬 핫시트 플레이를 할 수 있습니다.

## 온라인 멀티플레이

온라인 방 기능은 Firebase Cloud Firestore와 Firebase Hosting을 사용합니다.

1. Firebase 콘솔에서 Web App을 만들고 Firestore Database를 활성화합니다.
2. `firebase-config.example.js`를 복사해서 `firebase-config.js`를 만듭니다.
3. Firebase 콘솔의 Web App 설정값을 `firebase-config.js`에 붙여넣습니다.
4. `firebase-config.js`는 `.gitignore`에 포함되어 있으므로 커밋되지 않습니다.
5. 브라우저를 새로고침하면 `온라인 방`에서 방 만들기와 방 참가를 사용할 수 있습니다.

## 게임 규칙

1. 게임을 시작하면 플레이어는 50개의 코인을 지급받습니다.
2. 숫자 카드는 1~9까지 각 두 장씩 존재합니다.
3. 사칙 카드는 `+`, `-`, `×`, `÷` 각 두 장씩 존재합니다.
4. 숫자 조커와 사칙 조커가 한 장씩 존재합니다.
5. 숫자 조커는 원하는 숫자로 사용할 수 있습니다.
6. 사칙 조커는 원하는 연산기호로 사용할 수 있습니다.
7. 게임은 총 10라운드로 진행되며 각 라운드는 3페이즈입니다.
8. 첫 라운드는 가장 높은 숫자를 만드는 사람이 승리합니다.
9. 이후 라운드는 이전 라운드 승자가 다음 승리룰을 정합니다.
   - 1~50까지의 임의 숫자를 랜덤으로 정하고 가장 가까운 사람이 승리
   - 가장 큰 수를 만든 사람이 승리
   - 가장 작은 수를 만든 사람이 승리
10. 모든 코인을 잃은 플레이어는 해당 라운드에 직접 참여할 수 없습니다.
    - 대신 1페이즈에서 숫자 카드 2장을 받습니다.
    - 받은 2장의 카드는 즉시 공개됩니다.
    - 다른 플레이어는 공개 카드를 1코인에 선착순 구매할 수 있습니다.
    - 판매한 코인을 가지고 다음 라운드부터 다시 참여할 수 있습니다.
11. 전체 10라운드가 진행된 이후 가장 많은 코인을 보유한 플레이어가 최종 승자가 됩니다.

## 구현 파일

- `index.html`: 로컬/온라인 시작 화면, 대기실, 게임 테이블
- `styles.css`: 카드 테이블 UI와 반응형 레이아웃
- `app.js`: 게임 규칙, 로컬 플레이, Firebase 방 동기화
- `firebase-config.example.js`: Firebase 설정 템플릿
